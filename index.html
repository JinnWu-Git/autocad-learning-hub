2. AutoCAD 核心系統變數百科 [5]

FILEDIA

控制導航對話框。設為 1 顯示彈窗，設為 0 強制命令列輸入 [5]。

OSMODE

鎖點模式總和。常用 35 (端點+中點+圓心) 或 16383 (全開) [6, 7]。

PICKADD

控制選取累加。建議設為 2，以便連續選取多個物件而不消失 。

4. 如何成為 2D & 3D 高手？

2.D 精通路徑

  • **動態塊 (Dynamic Blocks)**：建立帶參數的萬用零件 。
  • **註釋比例 (Annotative)**：掌握不同比例出圖的一致性 。
  • **外部參考 (Xref)**：管理大型專案的層級結構 。

3.D 大師路徑

  • **布林運算**：熟練實體模型 (Solid) 的聯集與差集 [12, 13]。
  • **座標系 (UCS)**：靈活旋轉繪圖平面處理複雜面 。
  • **彩現渲染**：設定材質與燈光輸出擬真圖面 。

5. 常用 AutoLISP 函數實務 [12, 14, 15]

透過 AutoLISP 實現自動化繪圖，能將耗時數小時的工作縮短至秒級 。

核心函數對應表
setq設定變數值 [14] ssget過濾並選取物件 [16] defun定義新指令函數 command調用 AutoCAD 指令
實戰：全圖塊計數器 
(defun c:BlkCount (/ ss)
  (setq ss (ssget "X" '((0. "INSERT"))))
  (if ss
    (alert (strcat "偵測成功！目前圖面共計 " (itoa (sslength ss)) " 個圖塊。"))
    (princ "\n未發現任何圖塊。")
  )
  (princ)
)

8. AutoCAD 與 AI：設計的未來趨勢

生成式設計

**Project Bernini**：Autodesk 研究中的 AI 模型，能從文字提示快速生成具備功能性的 3D CAD 模型 。

神經 CAD (Neural CAD)

未來的 CAD 引擎將具備「物理推理」能力，能自動預測建築內部的管線配置與結構邏輯 。

智慧協作

**Markup Assist** 利用 AI 辨識手寫修正，自動將其轉化為數位指令（如 MOVE 或 DELETE） 。
