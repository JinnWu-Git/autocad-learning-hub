2. AutoCAD 核心系統變數庫

FILEDIA

控制檔案對話框。設為 0 時會隱藏彈窗，對新手最常見的崩潰痛點 [1]。

建議：1 (開啟)

OSMODE

鎖點模式位元碼總和。端點(1)+中點(2)+圓心(4)等 [2, 3, 4]。

常用：35 (基礎鎖點)

TRIMEXTENDMODE

2021+ 快速模式。設為 1 時，TRIM 不需預選邊界即可直接剪裁 [5, 6]。

建議：1 (快速)

5. 常用的 AutoLISP 函數實務

核心函數邏輯

  • `setq`：變數賦值 [9, 10]
  • `getpoint`：提示用戶選取圖紙座標點
  • `ssget`：建立選集（配合 "X" 為全圖選取） [11, 12]
  • `command`：在 LISP 中調用標準 CAD 指令 [13]

高手範例：自動區塊計數器 [14, 11]

(defun c:QuickCount ()
  ; 統計圖面中所有的 INSERT (圖塊) 物件 [14, 11]
  (setq ss (ssget "X" '((0. "INSERT"))))
  (if ss
    (alert (strcat "偵測成功！目前圖面共計 " (itoa (sslength ss)) " 個圖塊。"))
    (princ "\n圖面中沒有任何圖塊。")
  )
  (princ)
)
